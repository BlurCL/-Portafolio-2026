# Gestión del Proyecto - ConstruFácil

Esta carpeta contiene la documentación de gestión utilizada para respaldar la planificación, organización, control de avances y mejoras aplicadas durante el desarrollo del proyecto ConstruFácil.

## Contenido

- `01_PLAN_DE_TRABAJO.pdf`: resumen del trabajo por fases: inicio, desarrollo y cierre.
- `02_CARTA_GANTT_ConstruFacil.xlsx`: planificación del proyecto mediante carta Gantt.
- `03_RESPONSABILIDADES_EQUIPO.pdf`: distribución de responsabilidades por integrante.
- `04_CONTROL_DE_AVANCES.pdf`: avance del proyecto por evaluación y por componente.
- `05_RETROALIMENTACION_Y_MEJORAS.pdf`: hallazgos detectados durante pruebas y mejoras aplicadas.

## Nota

Esta carpeta no contiene código fuente. El código del sistema se encuentra en la carpeta `Producto` y la documentación formal de entrega se encuentra en `Documentacion`.
